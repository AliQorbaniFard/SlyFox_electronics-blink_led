This repository contains the code for blinking an LED on STM32 Blue Pill boards. 
The hardwares used in this project are the STM32F103C8T6 microcontroller and STLink V2 programmer.
The softwares used are MDK-ARM (Keil) and STM32CubeMX. 
A step-by-step video tutorial on how to do this project will be uploaded on YouTube soon and the link will be provided.
